Sudoku Game with GUI 

Description

This Python Sudoku Game project provides a graphical user interface (GUI) for playing Sudoku. The application allows users to play and enjoy Sudoku puzzles in a user-friendly environment.

Features

- Interactive GUI for seamless user experience
- Real-time validation of user inputs

Files

- **sudoku.py**: Contains the main logic for the Sudoku game.
- **gui.py**: Implements the graphical user interface using a GUI library.

Usage

1. Ensure that both `sudoku.py` and `GUI.py` are in the same directory.
2. Run the GUI file (`GUI.py`)
3. The GUI window will appear, allowing you to play Sudoku puzzles interactively.

Instructions

1. Click on a cell to select it.
2. Input a number (1-9) using the keyboard and confirm the number with "Enter" key.
3. The game will validate your input in real-time.
